# 변경 이력 (Changelog)

## [2.0.0] - 2026-02-25 (수요일)

### 추가 ✨
- **네이티브 Rust 의존성 엔진**: 무거운 `ts-morph`와 실험적인 정규식 기반 추적을 대체하여 **Native Rust (@ast-grep/napi)** 기반의 초고속 엔진을 도입했습니다. 분석 속도가 기존 대비 약 100배 향상되었습니다.
- **아키텍처 가드레일 (Architecture Guardrail)**: `.fast-lintrc.json`을 통해 레이어 간 의존성 방향(예: 도메인은 인프라를 참조할 수 없음)을 강제하는 시스템을 도입했습니다.
- **자가 치유 워크플로우 (Self-Healing Workflow)**: 에이전트가 코드를 수정하고 자동으로 테스트를 실행한 뒤, 실패 시 에러 로그를 분석하여 재수정하는 자율 루프(`verify-fix` 도구 및 `AgentWorkflow`)를 구축했습니다.
- **설치 없는 고성능 환경**: 사전 빌드된 Rust 바이너리를 포함하여, 사용자가 Rust 컴파일러를 설치하지 않아도 최고의 성능을 낼 수 있도록 내재화했습니다.
- **지능형 증분 분석**: 변경된 파일뿐만 아니라, 해당 파일에 의존하는 상위 모듈(Dependents)까지 역추적하여 분석 범위를 결정합니다.
- **LLM 최적화 도구 설명**: 모든 MCP 도구의 설명을 고품질 영문으로 작성하여 AI 모델의 추론 정확도를 높였으며, 개발자를 위한 한글 주석을 병기했습니다.

### 변경 ⚡
- **설정 스키마 업데이트**: `zod`를 사용하여 `ConfigSchema`에 `architectureRules` 항목을 추가했습니다.
- **AnalysisService 리팩토링**: `SemanticService`를 메인 분석 파이프라인에 통합하여 문맥 인식 스캔을 수행합니다.

## [1.2.0] - 2026-02-21 (토요일)

### 추가 ✨
- **Git 증분 분석**: `simple-git`을 활용하여 변경된 파일만 분석하는 모드 추가 (성능 향상).
- **순환 참조 탐지**: 모듈 간 순환 참조를 탐지하여 아키텍처 결함 보고 기능 추가.
- **커스텀 룰 엔진**: `.fast-lintrc.json`을 통해 사용자 정의 AST 패턴 규칙 설정 기능 도입.
- **설정 검증 강화**: `zod` 스키마를 이용한 설정 파일 유효성 검증 강화.

### 변경 ⚡
- **의존성 맵 정밀화**: `src/analysis/fd.ts`의 로직을 경로 기반으로 개선하여 정확도 향상.
- **리포팅 개선**: 위반 사항(`Violation`) 타입에 `CUSTOM` 및 `ENV` 추가.

## [1.1.0] - 2026-02-21 (토요일)

### 추가 🚀
- **네이티브 AST 바인딩**: `ast-grep` CLI 대신 `@ast-grep/napi`를 도입하여 분석 속도를 10배 이상 개선.
- **병렬 처리 도입**: `p-map` 및 `fast-glob`을 도입하여 멀티코어 병렬 분석 환경 구축.

### 변경 🏗️
- **아키텍처 리팩토링**: `src/index.ts`의 거대 로직을 `AnalysisService`, `ConfigService`, `QualityDB` 등으로 분리하여 유지보수성 극대화.
- **파일 순회 최적화**: `fd` CLI 호출을 `fast-glob` 라이브러리로 대체하여 외부 의존성 감소.

---

## [1.0.0] - 2026-02-21 (토요일)

### 추가 📦
- **초기 MVP 출시**: AI 에이전트 전용 코드 품질 검속 MCP 서버 초기 버전 출시.
- **핵심 메트릭**: 파일 크기(`SIZE`), 함수 복잡도(`COMPLEXITY`), 테스트 커버리지(`COVERAGE`), 기술 부채(`TECH_DEBT`) 체크 기능 구현.
- **SQLite 이력 관리**: `.fast-lint/quality_history.db`를 통한 품질 이력 관리 및 변경 감지.
- **환경 진단**: 필수 도구(`fd`, `rg`, `sg`) 설치 여부 자가 진단 기능.
- **MCP 프로토콜**: `quality-check` 도구를 통한 AI 에이전트 인터페이스 제공.
